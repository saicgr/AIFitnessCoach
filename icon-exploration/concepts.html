Zealova · App Icon

Three distinctive abstract marks

Every literal attempt (the feature collage, the mascot, the comet) failed the one test that matters: does it still read at 60 px? So these are abstract marks instead — each one a single bold form, hand-built as a precise vector (no AI image generator), in the brand orange #F97316 on a near-black background.

Each is shown big, then shrunk onto a home screen at 84 / 56 / 40 px — the real squint test — plus a light-background check. Pick the one that feels right and I'll refine and ship it.

My honest read — after rendering them

  • Ember renders as a coffee bean. A round disc plus one curved cut reads as coffee, full stop. It does not work.
  • Focus renders as a pair of headphones, and sits uncomfortably close to the Threads logo. It does not work.
  • Ascent is the only one that holds up — clean and legible down to 40 px. But it is, plainly, a bar chart. Usable, generic.

If you pick one

  1. Tell me the name. I'll refine that exact mark — weight, proportions, curve, and 2-3 variations of it so you can compare.
  2. I export it to PNG at every required size and drop it into ios/Runner/Assets.xcassets/AppIcon.appiconset + the Android mipmap-* set (one 1024 master → all sizes via flutter_launcher_icons).
  3. Done — it ships with the next build, replacing the orbit-Z that's live now.

If none of the three land, the honest fallback stands: a $30-150 designer (Fiverr / 99designs) or a tool like Looka. But these are real, controllable starting points — not generator guesses.
