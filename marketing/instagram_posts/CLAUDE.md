# Instagram Posts — Claude Instructions

Read `marketing/CLAUDE.md` first. This is for organic Instagram (Reels + carousels + grid posts). For paid creative, see `instagram_ads/CLAUDE.md`.

## What ranks in 2026 (verify with WebSearch every drafting session)

- **Reels dominate**: 7–15 second sweet spot. Average watch time + rewatches are the dominant ranking signal.
- **DM sends > shares > saves > comments > likes** — sends via DM are now the most heavily weighted distribution signal for Reels.
- **First 2.5–3.5 seconds** must land an emotional punch (curiosity / shock / humor). 25% of viral potential lives in those 3 seconds.
- **~50% of users watch on mute** — text on first frame is mandatory, not optional.
- **Caption earns the save**, the hook earns the view. Two different jobs.
- **3–5 hashtags** (verify the cap holds via fresh WebSearch each session — Instagram changed it in 2026 and may again). Mix: 1 large + 2 mid + 2 small/niche. Niche tags (<200k posts) currently carry more reach than broad ones. **Pick the specific tags fresh every session** — the small-tag landscape shifts fastest on IG. Query: `Instagram fitness app hashtag volume [Month] [Year] niche AI calorie tracker`.
- **Carousel posts** still get 2× engagement of single images for educational content. 7–10 slides, value-stacked.
- **Trending audio** is required for Reels. Original audio is hidden from non-followers.

**Run a fresh WebSearch each session** for trending Reels audio in fitness/tech-launch niches and the current Reels algo rules.

## Hook frameworks (Reels)

1. **The shared struggle** — "Still doing 100 crunches and seeing zero abs?" — fitness coach classic
2. **The contradiction** — "Snap a photo of your plate. That's it. No typing."
3. **The number drop** — "I logged 4,200 workouts in 14 days as a beta tester. Here's what surprised me."
4. **The before/after tease** — Show the chaos (typing into MyFitnessPal) → "But what if it was this?" cut to Zealova.

## Caption structure

```
[Hook line — 5–8 words, mirrors video hook]

[Context — 1–2 sentences, who built it / why]

[Value — the specific takeaway or feature shown]

[Save CTA — "Save this for later" or "Tag a friend who counts every almond"]

[3–5 hashtags on their own line at bottom]
```

Length: 125–220 chars total for highest engagement. Long captions only for carousel posts.

## Format rules

- **Vertical 9:16 only** for Reels. 4:5 for grid posts. Never 1:1 (looks dated in 2026).
- **Text on first frame** — 4–8 words, large legible font, contrast against video.
- **Captions hard-coded into video** (don't rely on auto-captions for the first 3 seconds).
- **No watermark from CapCut / TikTok** — Reels de-ranks watermarked content.
- **Link in bio only.** Body and caption can never link out.

## Carousel post template (10 slides)

1. **Cover**: bold claim or question. Big text, contrast.
2–8. **One idea per slide**: tight headline + 1 sentence. Same visual style across all.
9. **Recap** of the 7 ideas in a single list.
10. **CTA**: "Save this", "Follow for more", or "Comment 'X' for the link".

Carousel native template: keep 1080×1350 (4:5) for max screen real estate. Same font + color palette every slide.


## Anti-patterns

- Posting a Reel without text on first frame → 50%+ of views are mute, they leave.
- Generic "Watch this!" / "Wait for it..." hooks → 2026 algo flags as low-effort.
- Outbound links in caption → suppressed. Always "link in bio".
- 30 hashtags → spam signal. 3–5 niche tags only.
- Re-posting Reels from TikTok with watermark → de-ranked instantly.
- Posting at random times — the audience is largely B2C; weekday lunch + evening windows are non-negotiable.

## When the user asks for "an Instagram post"

1. Ask: Reel, carousel, or single? (Reel is the default — push back if user picks single.)
2. Run a parallel WebSearch batch (THIS SESSION):

   - `trending Instagram Reels audio fitness [Month] [Year]`
   - `Instagram fitness app hashtag volume [Month] [Year] niche AI calorie tracker`
3. Hook + script (first 3 seconds is the entire post).
4. Caption with save-CTA + the 3–5 chosen hashtags (this session's research) on the last line.
5. Append to `posts.md` with date, research log, format, plan, script verbatim.
