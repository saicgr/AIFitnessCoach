Zealova Flutter App Icon Bundle

Included:
- source/dark/app_icon_1024.png and source/light/app_icon_1024.png
- Android mipmap icon sizes for dark and light variants
- iOS AppIcon.appiconset for dark and light variants
- flutter_launcher_icons_dark.yaml and flutter_launcher_icons_light.yaml
- flutter_assets/assets/icons/ files for direct Flutter use

Recommended Flutter workflow:
1. Copy flutter_assets/assets/icons into your Flutter project assets/icons folder.
2. Add flutter_launcher_icons to dev_dependencies:
   dev_dependencies:
     flutter_launcher_icons: ^0.14.3
3. Copy either flutter_launcher_icons_dark.yaml or flutter_launcher_icons_light.yaml into your project root.
4. Run one of these:
   dart run flutter_launcher_icons -f flutter_launcher_icons_dark.yaml
   dart run flutter_launcher_icons -f flutter_launcher_icons_light.yaml

Notes:
- Google Play icon: android/<variant>/play_store/play_store_icon_512.png
- iOS marketing icon: included as 1024x1024 in each AppIcon.appiconset
- Light variant was generated by replacing the charcoal background with an off-white icon background while preserving the orange Zealova mark.
