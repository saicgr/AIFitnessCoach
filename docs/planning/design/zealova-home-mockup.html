Zealova v5 — 3 zones
Utility strip (white) → Today zone (warm) → Track zone (grey). Each zone is a labelled band that slides up like Google's sleep card. Denser than Google by design — it's a coach, not a tracker — but now grouped, so it reads as rooms.

How they compare: Google anchors with one hero and uses 2 zones. Zealova carries more (it's an action app) so it uses 3 — but the move that closes the rhythm gap is the same: full-bleed labelled bands with a background-color change between them, each with a slight rounded top so it reads as a stacked sheet. Quick actions + workout + nutrition all still land above the fold; water & fasting are the scroll reward.
