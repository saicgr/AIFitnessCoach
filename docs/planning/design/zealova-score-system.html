The one rule

applicable = contributors whose data exists today
  Train → only if a workout is scheduled today
  Fuel  → only if nutrition targets are set
  Move  → only if Health Connect is providing steps

score = round( Σ(weightᵢ · doneᵢ) / Σ(weightᵢ) × 100 )   over applicable i

No workout + no Health Connect → only Fuel applies → score = Fuel %. Honest, and the home turns into a setup checklist. No workout + low steps → score = Fuel + Move, low because you genuinely haven't done much — the score stays truthful, and the coach names the highest-leverage fix ("a 25-min walk is worth ~10 points"). Train never "helps" by being faked to 0 or 100 — it's simply in or out, and the rest rebalance.
