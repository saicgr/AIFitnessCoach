The smarter way — a pinned action strip (not "move it around")

You're right that repositioning never worked — any spot in the scroll gets scrolled past. The real fix is to take quick actions out of the scroll entirely. So: the quick actions become a slim pinned strip docked above the nav bar — one unified bottom panel. It never moves, the banner can't bury it, and it scrolls horizontally so it holds every action.

This also kills two other problems at once:
• Chat goes into the strip (first chip, accent-coloured "Coach") — so the floating button is gone, and with it the FAB-overlapping-the-Water-tile mess you saw.
• The strip holds 9 actions + a More chip, scrollable — Coach · Food · Scan Menu · Fridge · Photo Log · Weight Log · Measure · Mood · Recipe · Quick · More. Weight Log is a real chip now (no "More" stealing a visible slot); "More" sits at the scroll end and opens the full grid / My Space customization.

Point 2 (can't see quick actions with the banner on): solved permanently — pinned = always visible. Point 3 (smarter way): this is it — dock it, don't relocate it.

Cost: ~58px of permanent bottom chrome — but it replaces a ~130px scrolling section and the floating FAB, so the screen is actually calmer, not busier.
