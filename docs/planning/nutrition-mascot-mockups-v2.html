How it ships

These are hand-built SVG + CSS so you can judge the art and motion now. The approved one ships as a Lottie (lottie: ^3.1.0 is already in pubspec) embedded in hero_nutrition_card.dart, near the calorie-ring center. One driver value fullness = eaten/goal (0–1.3) cross-fades the mood states; a justAte trigger fires the hop on each log. Mood caption uses a variant pool (≥4 lines per state) so it never reads robotic.

Mood states: Hungry (0–15%) · Fueling (15–85%) · Nailed it (85–100%) · Over budget (>100%).
