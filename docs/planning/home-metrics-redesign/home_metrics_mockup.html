My recommendation: Direction C — compact metrics + full coach

  • Keeps the AI Coach full-size (message + 4-task carousel) — the moat — while metrics shrink to one glanceable summary card (big Today ring + 4 live tiles with graph overlays) you swipe for trends.
  • The FAB stays a pure AI Coach button. A Log button in the metrics card opens the quick-actions sheet (snap meal / scan menu / water / weigh in / start workout) and a Trends button opens your analytics — so coach and logging never share one control.
  • Bento (A) stays an option if you'd rather metrics be large and detailed up top; the rail (B) is the ultra-compact alt. Both remain in this file to compare.
  • My Space rows carry live mini-graphs + numbers (ring / sparkline / gauge / dual-line) matching each metric's chart type.
  • Tabs stay stable when you customize: Summary is always the Today ring; More = your enabled tiles; Trends = your chart/large widgets. Adding or removing metrics only reflows More/Trends — the tab structure and the pinned ring never change.
  • Three metric-card buttons — Log (opens the quick-actions sheet, now fuller since the standalone row is gone), Trends, and Start.

Your asks, mapped to real code

  • Bigger circle in A — done (132px segmented hero ring + legend).
  • Every widget can be Large + editable — phone 3: per-tile S / Wide / Large on all tiles, and the ⚙ opens an inline editor (color swatches, chart type Number/Ring/Bars/Line, date range 7d/30d/90d/1y).
  • Custom trends (2+ metrics) — phone 4 is the existing custom_trend_screen (metric multi-select + range + normalized chart + AI correlation). "Add to home" saves it as a Large widget; it also shows up in the metric list (the "Weight × Sleep TREND" row).
  • My Space — the editor is folded into My Space as a third tab (Metrics alongside the existing Sections + Discover), driven by home_sections_provider. "AI Coach" stays a section you can reorder.

  • Second metrics button — Start (workout), or something else (e.g. Customize)?
