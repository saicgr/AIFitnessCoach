Buddy & Bowl — now actually alive

Fixed: the old file disabled all motion if your Mac had "Reduce Motion" on. Removed. And the rig is now Duolingo-style — the buddy hops, sways, bobs its head, darts its eyes, blinks, waves, taps its feet, and its sprout wobbles, all continuously. 🍴 Feed = a celebration jump. Push it to Over and it goes woozy. Open in a browser.

Blob Buddy is rigged in parts: independent loops for hop, sway, breathing, head-bob, blinking, darting pupils, a waving arm, tapping feet, and a bouncing sprout — so it reads as a living character, not a still drawing. Hungry = pale & droopy. Goal = beaming + sparkles. Over = inflates, sweats, and wobbles woozily.
Cozy Bowl gently bobs, steam drifts faster, and it does a satisfying jiggle each time you feed it; over = food heaps over the rim and a dumpling tumbles out.

For production in Flutter, this exact feel is built with Rive (rive.app) — a real rigged character with a state machine driven by live calorie data. This HTML approximates the motion so you can pick the direction first.
