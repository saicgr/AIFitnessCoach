Changes in v8 — full day context + recovery guardrail
  • Conflict rows list every session already on that day (screen C/D, Tue: Mobility Yoga + 5K Tempo), not just one — so you see the whole day before deciding.
  • Over-exertion caution — a soft, non-blocking warning when a day stacks several sessions / a lot of time (screen A Thu, screen D Tue): encourages recovery or spreading load. Never hard-blocks. (Hooks into the existing recovery-aware signal.)
Changes in v9 — per-day only (removed "Start as")
  • Removed the Start As (Main/Extra) toggle and the separate Extra flow — you decide per day instead. On a day that already has workouts: Replace or Add alongside; free days just fill in.
  • Main / Extra are per-day results, not a program setting: the day's kept/first workout is the Main (drives Today); anything you "Add alongside" stacks as an Extra. A supplement like Core Finisher simply ends up Extra because you Add-alongside its days.
Changes in v6 — schedule filter
  • Filter bar on the full schedule (screen A): quick type chips (Strength / Cardio / Mobility / HIIT) + a ⛃ Filter chip that opens the filter sheet.
  • Filter sheet (screen H): multi-select by program (incl. AI + add-ons) and by workout type. "Clear all" resets to everything. (Build note: type comes from the workout's category/muscle tags; program from assignment_id — both already exist server-side.)
Changes in v5 — card appearance
  • Card appearance picker in Start Program (screen C) + per-card render options (screen G): Cover photo (default), Color (pick an accent), or Plain.
  • Background logic: curated programs default to their cover art → if no cover (custom-built plans), auto color gradient from the accent → never blank unless you choose Plain. Accent also colors the program's chip, banner, and Manage row, so multiple programs stay distinguishable.
  • Editable later from the program's Manage row, not just at start.
Changes in v4 (latest feedback)
  • Full Program view → screen F: every week (W1→W8) at once for your OWN program, with completion ✓ / current ● / rest. Closes the Hevy "see all upcoming weeks" gap. Opened by tapping a program (chip / banner / Manage row) and from "Open full schedule".
  • Multiple sessions per day + "Add-on" explained → screen A, Thu: Morning yoga ☀ → Afternoon Core 7-min 🕐 → Evening gym 🌙, ordered by time of day. "Add-on" simply means an extra session that doesn't replace your main one; the day groups sessions by time, and the Main session is the one that drives your home hero. (Build note: needs an optional time-of-day / order on workouts to sort multi-session days.)
