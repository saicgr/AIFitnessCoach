Why not the others (short version)

V1 framed mini-phone: charming, but the bezel admits “this is a picture of the app” — V1b makes the same proof feel live.  V2 / System B (conversational): the boldest idea here and worth keeping — but 11 chat-styled steps means typing animations, reaction copy per answer, and l10n for all of it; ship A first, A/B the coach-asks variant on 2–3 steps later.  V3 / System C (editorial): gorgeous brand unity, but giant Anton with zero product proof asks a cold install to trust typography; better fit for the website, paywall and share cards.  V4 / V5 / V6: strong single ideas — V5's goal-date is folded into V1b's demo (the “AUG 22” chip) and A3's sign-in card, which is the best of it without the overpromise risk.
