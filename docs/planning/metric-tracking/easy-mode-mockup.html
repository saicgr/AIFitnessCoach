Easy mode — multiple metrics

Easy is a single big poster number with steppers below. It is NOT a column table (that's Advanced). The natural way to add metrics here is one extra stepper per metric under the poster — the same pattern the LOAD stepper already uses for sleds.

Proposed rule: the exercise's primary metric stays the big poster (distance / time / reps); every other tracked metric (load, box height, calories…) becomes a labeled −/+ stepper in the row beneath it, plus a “+ metric” button. Steppers wrap to a second line past ~3.
