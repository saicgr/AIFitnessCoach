## Platform-Specific Notes

### Twitter/X
- Best times: 9am, 12pm, 5pm
- Use #buildinpublic #indiehacker
- Thread format works well for updates

### TikTok
- Best times: 7pm-9pm
- POV format performs well
- Keep under 60 seconds
- Use #fitnesstok #gymtok #tech

### Instagram
- Reels get more reach than posts
- Stories for daily updates
- Save best content for Reels

### LinkedIn
- Professional tone
- Weekly recap format
- Good for milestone announcements

### Reddit
- r/fitness - value-first, no promotion
- r/startups - build in public updates
- r/FlutterDev - technical posts
