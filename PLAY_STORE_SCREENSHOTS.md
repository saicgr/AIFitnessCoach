# FitWiz Play Store Screenshots Guide

## Setup
- **Tool**: [theapplaunchpad.com](https://theapplaunchpad.com)
- **Size**: 1080x1920px (portrait, 9:16)
- **Format**: PNG (24-bit, no alpha) or JPEG
- **Max file size**: 8MB per image
- **Template style**: Flo-inspired — bold caption top, single phone centered, colored background
- **App mode**: Dark mode ON

## Screenshot Details

### Screenshot 1: AI Coach Chat

**Caption**: "Your AI Coach. Ask Anything."
**Background**: Dark charcoal (#2D2D2D)
**Screen**: Chat screen

**What to capture:**
- 2-3 message bubbles showing coach response
- Coach avatar visible
- Quick action pills at bottom
- Input bar with camera/video buttons

**Best message to send**: `"I hit 225 on bench for 3 reps today!"`
- Short, produces a celebratory response
- Hard for AI to mess up
- Positive energy in the screenshot

**Backup**: `"What should I focus on today?"` — references your schedule, looks personalized.

---

### Screenshot 2: AI Food Parser

**Caption**: "Type Any Meal. Instant Nutrition."
**Background**: Warm orange (#F97316)
**Screen**: Log Meal sheet

**What to capture:**
- Parsed result AFTER AI analysis
- User typed: `"chicken biryani with raita and a mango lassi"`
- 3-4 food items with name, calories, macros (P/C/F)
- Meal type selector showing "Lunch"

**How to get this state:**
1. Nutrition tab → "+" → Log Meal
2. Select "Lunch"
3. Type the meal, tap analyze
4. Screenshot the result cards

---

### Screenshot 3: AI-Generated Workout

**Caption**: "Every Exercise. Chosen For You."
**Background**: Soft gray (#6B7280)
**Screen**: Workout detail

**What to capture:**
- Exercise list with sets/reps visible
- At least one superset indicator
- "WHY THESE EXERCISES?" reasoning expanded
- Stats row (duration, exercise count)

**How to get this state:**
1. Generate a workout from home screen
2. Open workout detail
3. Expand "More Info" → "Why These Exercises?"
4. Screenshot

---

### Screenshot 4: Progress Dashboard

**Caption**: "Track Every Rep. See Every Gain."
**Background**: Deep green (#166534)
**Screen**: Stats overview

**What to capture:**
- GitHub-style activity heatmap with green squares
- Streak counter
- PR highlights
- Volume or strength chart trending up

**Pre-condition**: Need 20+ workouts logged for a good-looking heatmap.

---

### Screenshot 5: Exercise Preferences

**Caption**: "It Learns What You Love."
**Background**: Muted gold (#B45309)
**Screen**: Exercise preferences

**What to capture:**
- Three sections visible:
  1. Staple exercises (gold stars) — Bench Press, Squat, Deadlift
  2. Avoided exercises (red X) — Behind Neck Press, Smith Machine Squat
  3. Exercise queue (blue icons) — Cable Flyes, Face Pulls

**How to get this state:**
1. Settings → Training Preferences → "Favorites, avoided & queue"
2. Add exercises to each section
3. Screenshot showing all three sections
